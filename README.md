## 📖 Project Overview

This project presents an interactive Tableau dashboard designed to analyse heart disease-related risk factors using a public health dataset.

The dashboard enables users to explore disease prevalence, smoking behaviour, BMI patterns, geographic distribution of heart attack cases and perform What-If analysis through interactive parameters and filters.

---

## 🎯 Business Problem

Healthcare organisations collect large volumes of health data, but identifying meaningful patterns from raw datasets can be challenging.

This dashboard transforms complex health data into interactive visualisations that help users:

- Identify disease prevalence
- Analyse heart attack risk
- Compare demographic groups
- Explore lifestyle risk factors
- Simulate future risk scenarios

## 💡 Key Insights

- Regular smokers showed higher heart attack rates than non-smokers.
- Heart attack risk increased with age.
- BMI patterns varied across diseases and physical activity levels.
- Geographic analysis highlighted variations in heart attack cases across states.
- Interactive What-If analysis allows users to simulate changes in heart attack risk.
