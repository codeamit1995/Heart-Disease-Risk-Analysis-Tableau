# 🩺 Disease Analysis Dashboard | Tableau

An interactive Tableau dashboard designed to analyse heart disease-related lifestyle and demographic risk factors. The dashboard enables users to explore disease prevalence, smoking behaviour, BMI patterns, geographic distribution of heart attack cases, and perform What-If analysis using interactive parameters.

# 📌 Project Overview

Healthcare datasets contain valuable information that can help identify disease trends and support better decision-making. This project transforms raw health survey data into an interactive Tableau dashboard that allows users to explore relationships between demographic characteristics, lifestyle factors, and heart attack risk.

The dashboard combines multiple visualisations into a single interactive interface, making it easier to identify trends, compare population groups, and simulate changes in heart attack risk using What-If analysis.

---

# 🎯 Business Problem

Healthcare professionals and decision-makers need intuitive tools to explore health data efficiently. Analysing raw datasets can make it difficult to identify patterns across multiple variables.

This dashboard addresses that challenge by providing an interactive visual analytics solution that enables users to:

* Identify the most common diseases
* Compare heart attack rates across smoking groups and gender
* Analyse BMI patterns by disease and physical activity
* Explore geographic distribution of heart attack cases
* Simulate changes in heart attack risk using What-If analysis

---

# 🔍 Key Insights

* Arthritis and Depressive Disorder were the most common diseases in the dataset.
* Regular smokers showed a higher heart attack rate than non-smokers.
* Heart attack risk increased steadily with age.
* BMI varied across diseases and physical activity levels.
* Geographic differences highlighted regions with higher simulated heart attack cases.
* What-If analysis demonstrated how changes in risk factors could influence heart attack rates.
